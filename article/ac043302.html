安装依赖

  • node
  • npm
  • vue cli

官网

搭建命令

完成上述工具安装以后，可以用node -v npm -v vue -version 查看版本

  1. vue create project-name 到你想创建项目的位置使用命令
  2. cd project-name
  3. npm run serve 启动vue服务，稍后可在http://localhost:8080/查看

文件层次说明

  • public
    静态资源文件夹，和assets不用，public不会被webpack进行打包，使用时，路径要加绝对值。
  • src
    项目核心区域，所有开发文件，包括静态资源，组件等
  • assets 静态资源
  • components 组件
  • views 公共文件,需要自己建,以各个独立页面为主
  • router 路由（项目配置路由）
  • app.vue 根组件
  • main.js 入口文件

组件传值

以父组件app.vue和子组件HelloWorld.vue为例
父组件通过v-bind自定义属性传入值,子组件通过props接受对应参数
props:[]里面要将v-bind中的-变成驼峰
驼峰命名 即value-num == valueNum

  • 父->子

props也可以变成obj和arr
即props：{
valueNuma:String,
valueNumb:{
type:Number,
required:true, // 必须,通常不会和下面default一起使用
default:500 //父组件没传时会用这个替代
validator: function(value){ // 格式校验处
return value > 1
}
}
}
