云服务器和云数据库使用

发表于2021-02-04|更新于2021-02-04|服务器阿里云数据库PolarDB
|字数总计:1.6k|阅读时长:5分钟|阅读量:|评论数:

前言

我们在学习时候，经常会产生一些forms，需要别人进行填写，并且希望收到回馈，而发出的静态页面不能够记录别人的message，所以，需要购买一台云服务器和云数据库，在云端为我们服务。

  • 什么是云服务器:简而言之，就是在云端，一个虚拟的环境，有一台电脑，在这个电脑上可以do what you love do
  • 什么是云DB：同上，有一个云仓库在云端，是虚拟的，但是也有大小，可以存放你想存的东西，并通过code对里面数据进行增删改查等一系列基操。

阿里云ECS服务器

  • 作为一个白嫖砖家，首选阿里云免费使用一个月的ECS服务器
  • 当然，可以各大平台都去“探索一下”，一般来说，都有新用户体验。
  • 购买流程完成之后，进入我的实例。

购买流程及其配置

  1. 基础配置
    1. 进入官网，找到ECS服务器购买，1核2G永远的神。
    2. 可以选择按量付费或者包年包月。
    3. 地区和区域，离你近就行，随便选，访问速度有点差异。

    4. 任意镜像+高效云盘40G(后续需换系统至少40G)，下一步
  2. 网络和安全组
    1. 选择按使用流量 设置为80M
    2. 点击确认订单
  3. 进入管理控制台页面
  4. 设置安全组
    1. 进入点击实例ID或者配置规则，进入配置
    2. 百度自己的IP地址，复制

    当你更换冲浪的地方时候，还想要通过BT面板对你的服务器进行管理，需要把下面的8888这里的ip换成你当前所在地

    1. 添加8888端口号入口放开为你的第二步的ip。授权对象，源填ip，保存
    2. 将80端口号放开入口。授权对象，源填 0.0.0.0/0 保存
  5. 重置操作系统和镜像
    1. 回到控制台页面，先停止实例。
    2. 点击更多-云盘和镜像-更换操作系统
    3. 选择宝塔面板
    4. 安全设置，选择自定义密码
    5. 设置账号和密码，确认订单。
  6. 链接服务器，得到BT面板登录信息。
    1. 在实例页面，管理旁边 点击远程连接
    2. 账号密码填第五步的账号密码
    3. 进入页面如下，输入bt default
    4. 复制外网面板地址，到浏览器打开。
    5. 填入下面username和password
  7. 搭建服务器环境——tomcat
    1. 在左下方软件商店搜tomcat,选最新版本tomcat9
    2. 下载后点设置进行配置修改，只需要改端口号即可。
    3. 重启服务！
  8. 部署项目

    • 什么？这都不会？汤姆猫都没玩过？看你右边没，有堵墙，撞上去。

阿里云数据库——PolardDB

  • 兼容MYSQL,实质一样。
  • 和服务器一样，先嫖着~

你也可以在BT的软件商店中装MYSQL，个人喜好。

购买流程及其配置步骤

  1. 进入官网，找到云数据库
  2. 立即购买，选择两核8G入门级
  3. 上面兼容性，看个人喜好，可选Oracle
  4. 勾选协议，付费，支付成功。
  5. 进入控制台，地区可能要切换回来
    1. 点击集群id
    2. 左侧 配置与管理-集群白名单
    3. 新增白名单，名称随意，填入百度查来的ip

当你更换冲浪的地方时候，还想要对你的仓库进行管理，需要把白名单这里的ip换成你当前所在地

  1. 设置安全组
    • 在第5步的集群白名单页面，单击选择安全组或目标安全组操作栏中的配置按钮来更改安全组设置。
    • 在弹出面板中，选中一个即可（必须先有服务器实例，才会有选项！）
  2. 创建数据库高权限账号
    1. 回到PolarDB控制台。
    2. 在控制台左上角，选择集群所在地域。
    3. 找到目标集群，单击集群ID。
    4. 在左侧导航栏中，选择配置与管理 > 账号管理。
    5. 单击创建账号。
    6. 在创建账号面板，只需要设置账号密码，即你的数据库账号密码，连接时要用。
  3. 申请公网地址
    1. 有了数据库账号密码，还需要有公网地址
    2. 回到PolarDB控制台。
    3. 在控制台左上角，选择集群所在地域。
    4. 找到目标集群，单击集群ID
    5. 在基本信息页面的链接地址区域
    6. 申请
  4. 使用DMS连接集群
    • 也可以本地连接云数据库，有了账号密码。
    • 在控制台，基本信息页面右上角，单击登录数据库
    • 填入账号密码，测试链接，授权即可。
    • 进入DMS，看左侧已连接的实例,里面有默认系统表，不用动。
    • 数据库的一系列操作？okok hold on
